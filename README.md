# 📂 File Organizer

A Python-based utility that automatically organizes files in a directory into categorized folders such as Images, Documents, Videos, Music, and more.

This tool helps users clean up cluttered folders (like Downloads or Desktop) by scanning files, identifying their types based on extensions, and moving them into appropriate folders. It also ensures that duplicate file names are handled safely by renaming files to avoid overwriting.

## 🚀 Features

* 📁 Automatic file sorting by type
* 🔍 Detects file extensions and categorizes them
* 🛡️ Prevents overwriting by renaming duplicates
* ⚡ Fast and lightweight CLI tool
* 💻 Works on any operating system

## 🎯 Use Case

Perfect for organizing messy folders such as Downloads, Desktop, or project directories in just one command.

## 🧠 Tech Stack

* Python
* OS & File Handling Modules (os, shutil)
